# Python Network Scanning and Monitoring Tool

A Python-based network scanner that checks whether devices are reachable on the network using ICMP Ping requests. The tool records each target's status (UP/DOWN/TIMEOUT) and exports the scan results to a JSON file.

## Features
- checks multiple targets
- supports windows and linux/MAC ping commands
- records UP/DOWN/TIMEOUT scan results
- saves scan results, including timestamps in JSON format
- uses modular structure with separate components

## How to use this?
1) clone the repository
2) add target IP addresses or hostnames to the target list file
3) run main.py
4) navigate to output folder to view results of the network scan
